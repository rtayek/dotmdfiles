# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

A collection of prototype `.md` template files for software projects. Each file covers a distinct concern and is meant to be copied and customized per project. No build, compile, or run step — documentation only.

## Key files

| File | Purpose |
|------|---------|
| `agents.md` | Rules governing what agents may do freely, must ask about, or must never do |
| `persona.md` | AI agent identity, communication style, and tone |
| `coding-style.md` | Naming, formatting, and language-specific rules (Java 25+, C/C++, NASM) |
| `design.md` | Architectural principles, design goals, and engineering guidelines |
| `patterns.md` | Catalog of OO, architectural, enterprise, and test patterns |
| `README.md` | Project overview and usage instructions |

## Conventions

- Follow `coding-style.md` for any code examples embedded in the templates.
- Follow `design.md` for all architectural decisions.
- Follow `agents.md` for rules on what requires approval before acting.
- Follow `persona.md` for communication style.
